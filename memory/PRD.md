# Honey Royal Vip - Landing Page PRD

## Original Problem Statement
Crear una página web de Honey Royal Vip, un estimulante sexual natural (miel energética). Landing page única con scroll, estilo moderno y limpio. Solo botón de WhatsApp para pedidos.

## User Personas
- **Target Audience:** Hombres adultos en España buscando productos naturales para mejorar rendimiento sexual
- **Primary Action:** Contactar vía WhatsApp para realizar pedido

## Core Requirements (Static)
- Landing page única con scroll suave
- Diseño moderno y limpio (no oscuro/lujoso)
- Botones de WhatsApp para pedidos (+34 635 11 75 98)
- FAQ con instrucciones de uso
- Precios: 1 caja (31,99€), 2 cajas (53,99€), 3 cajas (76,99€)
- Información de envío: 48-72h, contrareembolso

## What's Been Implemented (Dec 2025)
- ✅ Hero Section con imagen de producto y CTA de WhatsApp
- ✅ Benefits Section con 6 beneficios del producto
- ✅ Galería de producto con 4 imágenes del usuario
- ✅ Pricing Section con 3 packs (destacando Pack Ahorro)
- ✅ Shipping Section con información de envío discreto
- ✅ FAQ Section con 6 preguntas frecuentes (incluye uso)
- ✅ Footer con contacto
- ✅ Botón flotante de WhatsApp
- ✅ Diseño responsive (mobile, tablet, desktop)
- ✅ Animaciones suaves y efectos hover

## Tech Stack
- Frontend: React + Tailwind CSS + Shadcn UI
- Backend: FastAPI (no usado para esta landing)
- Fonts: Outfit (headings) + DM Sans (body)
- Colors: Amber/Gold (#F59E0B), WhatsApp Green (#25D366)
